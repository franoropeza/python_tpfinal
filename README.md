# Trabajo Practico Final - Python

Este repositorio contiene el trabajo practico final del curso de Python.

## Archivos principales

- `TP_Final_Loteria_Salta_GitHub.ipynb`: notebook principal del trabajo.
- `dataset_loteria_salta.csv`: dataset utilizado para el analisis.
- `requirements_notebook.txt`: librerias necesarias para ejecutar la notebook.

## Temas analizados

- juego favorito segun cantidad de apuestas
- juego con mayor y menor recaudacion
- horario favorito de juego
- actividad por genero estimado y rango etario estimado
- relacion entre recargas y volumen apostado
